# Release v18.0.0 — Peaufinage Pédagogique & UX Module Formation

## Nouvelles fonctionnalités

### 1. Lecteur de Leçons enrichi
- **Bloc-notes privé intégré** : zone de prise de notes persistante par leçon, visible dans la sidebar
- **Feedback de fin de leçon** : notation par étoiles, niveau de difficulté ressenti, commentaire libre (Indicateur Qualiopi I32)
- **Quiz intégré** : questions avec lettres A/B/C/D, feedback coloré sémantique (vert = correct, orange = sélectionné)
- **Documents multi-format** : PDF, vidéo MP4 avec boutons Télécharger / Plein écran / Lire

### 2. Page Mes Récompenses (nouvelle)
- **Hero banner** : total de points, nombre de badges et certificats
- **Certificats téléchargeables** : un par bloc RNCP, avec barre de progression 100% et cachet LFTG
- **Collection de badges** : 10 badges (6 débloqués, 4 verrouillés avec filtre grisé)
- **Classement de promotion** : top 5 avec médailles or/argent/bronze

### 3. Schéma Prisma étendu
7 nouveaux modèles : `CourseObjective`, `CoursePrerequisite`, `LessonObjective`, `LessonFeedback`, `LearnerNote`, `Certificate`, `Badge`

### 4. Navigation mise à jour
Nouveau lien "Mes Récompenses" avec icône Trophy dans la section Formation de la sidebar
